# Manny

High order component for rendering the <Link to="/docs/manny">manny</Link> 
package with some configurable options specific to the mannys.game NFTs. 

<h2 style={{ marginTop: 0}}>Usage</h2>

<Code language="js">
{`<Canvas>
  <Manny
    animation="bored"
    onLoad={() => alert('lol')}
    position={[0, 45, 0]}
  />
</Canvas>`}
</Code>

## Props

### accessories

<Code language="js">
{`object // if provided, loads accessories on manny with useAccessories`}
</Code>

### animation

<Code language="js">
{`string // loads a valid animation from constants.LIBRARY`}
</Code>

### onLoad

<Code language="js">
{`function // optional callback when model has loaded`}
</Code>

### paused

<Code language="js">
{`boolean // if true, pauses current animation`}
</Code>

### position

<Code language="js">
{`[integer] // x/y/z coords for THREE.Object3D.position`}
</Code>

### rotation

<Code language="js">
{`[integer] // x/y/z coords for THREE.Object3D.rotation`}
</Code>

### scale

<Code language="js">
{`integer // scale to use for all values in THREE.Object3D.scale`}
</Code>

### textureUrl

<Code language="js">
{`string // unique NFT texture URL to use on manny model`}
</Code>

### tokenId

<Code language="js">
{`integer // token id of manny NFT`}
</Code>

### useTextureHD

<Code language="js">
{`boolean // if true, textureUrl will be swapped with a high resolution version`}
</Code>
