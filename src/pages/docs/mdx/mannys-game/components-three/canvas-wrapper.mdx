# Canvas Wrapper

Adds a `@react-three/fiber` `<Canvas>` tag with optimal mannys.game settings + lighting.

<h2 style={{ marginTop: 0}}>Usage</h2>

<Code language="js">
{`<CanvasWrapper>
  {sceneChildren}
</CanvasWrapper>`}
</Code>

### canvasProps

<Code language="js">

</Code>

### children

{`any // rendered inside the Canvas`}
