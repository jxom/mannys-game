# /download-token

## /:token-id

Download a .zip file containing the 3D model of the specified token id.

<h3 style={{ marginTop: 0 }}>Usage</h3>

Requires a `POST` request with a `body` that contains a `sig` and `token`.

<Code language="js">
{`const tokenId = 1;\nfetch(\`/api/download-token/\$\{tokenId\}\`, {
  method: 'POST',
  mode: 'cors',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    sig: signature, // message signed by owner of token id
    token: tokenId, // token of message signer
  }),
})`}
</Code>

### Return Value

A base64 encoded .zip that should be downloaded to users local machine.
