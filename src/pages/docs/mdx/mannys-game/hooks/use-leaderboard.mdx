# useLeaderboard

Hook for getting leaderboard data.

<h2 style={{ marginTop: 0 }}>Usage</h2>

<Code language="js">
{`const { data, load } = useLeaderboard();
// ...
load();
// ...
{data.map(gamer => <Gamer {...gamer} />)}`}
</Code>

## Return Value

<Code language="js">
{`{
  data: [gamers], // array of gamer objects sorted by rank descending
  load: function, // takes address as argument and loads achievements
  loading: boolean, // true if achievements request is loading
  error: string, // error message if request fails
}`}
</Code>
