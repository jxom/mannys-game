# Button

Used for rendering buttons across the site.

<h2 style={{ marginTop: 0}}>Usage</h2>

<Code language="js">
{`<Button color="yellow" onClick={() => alert('lol')}>
  \{\'Click Me\'\}
</Button>`}
</Code>

## Props

### bgColor

<Code language="js">
{`string // specific background color`}
</Code>

### color

<Code language="js">
{`string // border/bg hover color`}
</Code>

### children

<Code language="js">
{`React.Component // rendered inside button`}
</Code>

### className

<Code language="js">
{`string // added to outer container`}
</Code>

### large

<Code language="js">
{`boolean // if true, rendered with more padding`}
</Code>

### onClick

<Code language="js">
{`function // onClick callback`}
</Code>

### style

<Code language="js">
{`object // style object applied to outer container`}
</Code>

### textColor

<Code language="js">
{`string // adds text color css class`}
</Code>
