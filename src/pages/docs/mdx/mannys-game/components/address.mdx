# Address

Shows a given `account` address with some configurable functionality. If 
`account` is not provided, a [ConnectKit](https://docs.family.co/connectkit) Connect button will render instead.

<h2 style={{ marginTop: 0}}>Usage</h2>

<Code language="js">
{`<Address 
  account={{ 
    address: "0xlol", 
    ens: { name: "lol", avatar: "lol.png" } 
  }} 
  link="https://lol.com"
/>`}
</Code>

## Props

### account

<Code language="js">
{`{
  "address": string, // wallet address of account
  "ens": object, // { name: string, avatar: string }
  "isConnecting": boolean // true if user is connecting wallet
}`}
</Code>

### color

<Code language="js">
{`string // Optional color to render address in`}
</Code>

### isYou

<Code language="js">
{`boolean // if true renders "you" string`}
</Code>

### link

<Code language="js">
{`string // if provided, renders <a href={link}>`}
</Code>

### onClick

<Code language="js">
{`function // if provided, renders <span onClick={onClick}>`}
</Code>

### size

<Code language="js">
{`string // "short" or "long", determines styling`}
</Code>
