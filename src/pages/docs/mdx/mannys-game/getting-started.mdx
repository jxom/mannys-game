# Getting Started

<a href="https://github.com/mannynotfound/mannys-game" target="_blank">mannys.game</a>
is the repository holding the front end code for this site. If you'd like 
to build upon mannys.game, you can reference these docs.

## Prerequisites

- [Node/NPM v16](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

## Installation

Clone the mannys-game repository and install dependencies.

<Code language="bash">
  {`git clone https://github.com/mannynotfound/mannys-game
cd mannys-game && npm install`}
</Code>

## Quick Start

<Code language="bash">
  {`cp .env.example .env`}
</Code>

Fill out your .env variables and then:

<Code language="bash">{`npm start`}</Code>

You should now be able to visit the app at `http://localhost:3000`.

You may notice some pages that depend on server data wont load correctly, so to fix this
run the dummy server in a new terminal:

<Code language="bash">{`npm start-server`}</Code>
