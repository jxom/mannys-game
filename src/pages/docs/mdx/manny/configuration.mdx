## modelPath

Url for loading manny model - defaults to a path hosted by manny himself but useful if
using your own custom model or hosting solution.

## textureUrl

Url for loading manny skin texture - defaults to the basic manny texture but useful if
using a custom texture such as an NFT manny.

## onLoad

Function is called after manny model has been loaded.

### paths

Object where keys represent animation name, and values represent animation asset url to load.

### active

Key of active animation to play.

### fadeIn

Duration in seconds to transition in new animation.

### fadeOut

Duration in seconds to transition out new animation.

### paused

Toggle animations pause state.
